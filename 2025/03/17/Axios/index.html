Axios

Axios 是一个基于 Promise 的 HTTP 客户端，广泛用于浏览器和 Node.js 环境。它支持请求/响应拦截、取消请求、自动转换 JSON 数据等特性，简化了 HTTP 请求处理。

在发起请求时，可以传递一个配置对象来定制请求。常见的配置选项包括：

  • url：请求的 URL。
  • method：请求方法，如 GET、POST、PUT、DELETE 等。
  • params：URL 参数，用于 GET 请求。
  • data：请求体数据，用于 POST、PUT 等请求。
  • headers：请求头信息。

axios 的响应对象包含以下属性：

  • data：服务器返回的数据。
  • status：HTTP 状态码。
  • statusText：HTTP 状态信息。
  • headers：响应头信息。
  • config：请求时的配置对象。
